🚀 Quick Setup Guide
1
Python dependencies install karo (sirf ek baar)
pip install flask flask-cors selenium webdriver-manager openai pymupdf
2
Local server start karo
python cibil_automation.py  ya  CIBIL_LAUNCHER.bat double-click karo
Server port: 5000 (upar URL change kar sakte ho)
3
OpenAI API Key daalo — App 1 → Settings
Key aapke browser (localStorage) mein save hogi — kisi server pe nahi.
✓ Server online ho to AI wahan process hogi (PDF support + faster)
✓ Server offline ho to browser directly OpenAI API call karega (images only)
4
App 2 mein CIBIL Login karo
Username + Password daalo → “Remember” checked rakho → Login karo.
🔒 Credentials sirf aapke browser (localStorage) mein save hote hain — kisi remote server pe nahi
🔒 CIBIL Credentials:  Sirf aapke browser ke localStorage mein saved — koi remote server nahi, koi cloud nahi
🔑 OpenAI API Key:  Browser localStorage mein saved — direct mode mein browser se call hoti hai, kisi third party ko nahi jaati
☁ App 1 (Online Mode):  Form fill, download, aur AI extraction — sab kuch bina local server ke bhi kaam karta hai
🖥 App 2 (Local Required):  Selenium automation ke liye local Python server zaroori hai — jo aapke hi PC pe run hota hai
